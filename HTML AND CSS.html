What is HTML5?

HTML5 (HyperText Markup Language, version 5), is the standard markup language for documents designed to be displayed in a web browser. It is the basic element for a website, the foundations.

What is CSS3?

CSS3 (Cascading Style Sheets, version 3), is the language used for the design of the website, focus on the visual part of the document.
